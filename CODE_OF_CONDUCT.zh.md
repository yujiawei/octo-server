# 贡献者公约 行为准则（简体中文）

## 我们的承诺

作为社区成员、贡献者与 leader，我们承诺：无论年龄、体型、身心能力、
民族、性别认同与表达、经验水平、教育程度、社会经济地位、国籍、外貌、
种族、宗教信仰、性取向如何，每一位参与者都能享受无骚扰的社区体验。

我们承诺以开放、欢迎、多元、包容、健康的方式行动与互动。

## 行为准则

有助于建设积极环境的行为包括：

- 展现对他人的同理心与善意
- 尊重不同的观点与经验
- 坦然给予与接受建设性反馈
- 为自己的错误承担责任并向受影响者道歉
- 关注对整个社区的最优解，而不仅仅是个体

**不可接受的行为**包括：

- 使用带性暗示的语言或图像，以及不当的性关注或追求
- 钓鱼、侮辱、人身攻击、政治攻击
- 公开或私下的骚扰
- 未经允许公开他人私人信息
- 其他在专业场合明显不当的行为

## 执行职责

社区 leader 有责任厘清并执行行为准则，并对任何他们认为不适当、
具威胁性、冒犯性或有害的行为采取公平、合理的纠正措施。

## 适用范围

本行为准则适用于所有社区空间，以及个人在公共空间中代表社区时的行为。

## 举报

任何辱骂、骚扰或其他不可接受行为，可通过邮件向社区 leader
**conduct@mininglamp.com** 举报。所有举报将被及时、公正地审核与处理。
